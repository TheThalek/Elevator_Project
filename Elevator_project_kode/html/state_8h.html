void state_init ()
 Initializes the elevator. If the elevator is in an undefined floor, when the program starts, it moves up until it reaches a defined floor.
 
void read_stop_signal ()
 Reads the stop-signal button and determines the new state. If current_postion() is on undefined floor, current_state = StopShaft. If current_postion() is on defined floor, current_state = StopFloor.
 
void state_idle ()
 Elevator stands still on defined floor with doors closed, Continuously look for new orders. If new order, sets start direction, before changing the state to Move.
 
void state_wait ()
 Opens the door, when the timer expires, closes door. If obstruction signal is active, the door stays open. If the timer expires when the current_direction = 0, the state goes to Idle. Otherwise it goes to Move.
 
void state_stop_floor ()
 Stop button is activated when the elevator is on defined floor. The stop light is turned on and the doors are opened. All pevious orders are cleared. When the stop-button is released the stop light turns off. The door closes after the timer is expired.
 
void state_stop_shaft ()
 Stop button is activated when the elevator is on undefined floor. The stop light is turned on. All previous orders are cleared. When the stop-button is released the stop light turns off and the elevator stands still until a new order is placed. A new direction is chosen. New state = move.
 
void state_move ()
 Elevator is moving. The movement is decided by queue_manager();.
 
void state_run ()
 Implementation of the state machine. The lights and orders are updated. The stop-button is checked. The chosen switch case is run.
